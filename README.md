# Nuxt 3 + TailwindCSS + DaisyUI - Starter template

## Setup

# npm

npm install
npm update
npx nuxi upgrade

## Development Server

Start the development server on http://localhost:3000

```bash
npx nuxi dev
```

## Production

Build the application for production:

```bash
npx nuxi build
```

Locally preview production build:

```bash
npm nuxi build
```
